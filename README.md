# DevNaRTC

DevNaRTC é um guia simples e direto ao ponto focado na Reforma Tributária do Consumo, feito de desenvolvedor para desenvolvedor. 

Ele nasceu com o objetivo de descomplicar os impactos reais da reforma no nosso código: como lidar com o IVA Dual (IBS e CBS), o novo motor de cálculos "por fora", as regras de Split Payment na hora do checkout e os novos formatos de XML das notas fiscais.

## Como rodar o projeto

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Rode o servidor de desenvolvimento:
   ```bash
   npm run start
   ```

3. Acesse `http://localhost:4200` no seu navegador.
